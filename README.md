
    
O site "Agro Forte, Futuro Sustentável" foi desenvolvido com o objetivo de apresentar a importância da sustentabilidade no agronegócio. A página inicial possui um visual moderno e atrativo, com uma imagem de fundo relacionada ao campo e à produção agrícola. O menu de navegação permite que o usuário acesse facilmente as diferentes seções do site. Na área principal, é apresentado o tema central, destacando a necessidade de equilibrar a produção de alimentos com a preservação ambiental. O projeto explica como a agricultura sustentável contribui para o desenvolvimento econômico e para a conservação dos recursos naturais. Também são mostradas práticas sustentáveis importantes para o campo. Entre elas estão o uso inteligente da água, a preservação ambiental e a utilização de tecnologias modernas na agricultura. Cada prática é apresentada em cartões informativos de fácil leitura. O site conta ainda com um botão interativo que exibe uma mensagem educativa sobre sustentabilidade. Outro recurso implementado foi um contador animado que simboliza a quantidade de árvores preservadas, reforçando a importância da conservação ambiental. O design foi desenvolvido utilizando HTML para a estrutura, CSS para a estilização e JavaScript para as interações. Além disso, o layout é responsivo, adaptando-se a computadores, tablets e celulares. As cores predominantes foram escolhidas para remeter à natureza, utilizando tons de verde e elementos visuais ligados ao meio ambiente. Dessa forma, o site busca conscientizar os visitantes sobre a necessidade de produzir mais alimentos sem comprometer os recursos naturais das futuras gerações.
